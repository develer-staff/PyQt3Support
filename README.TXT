================================================================
         PyQt3Support - PyQt4.Qt3Support Python bindings
================================================================

----------------------------------------------------------------
What is this?
----------------------------------------------------------------

PyQt3Support is a porting to Qt3Support of some of the
PyQt3 widgets.

----------------------------------------------------------------
Contents
----------------------------------------------------------------

This package contains:

 - the autogenerated sip/Qt3Support/ tree, containing a
   patched version of the sip files from PyQt3

 - a patched version of the qt3to4 xml configuration file,
   q3porting.xml

 - a patch to other PyQt4 sip files and to the configure.py
   script

----------------------------------------------------------------
Presequisites
----------------------------------------------------------------

- Check the installed Qt version is >= 4.3.0
  (Ubuntu feisty with backports provides 4.3.1)

- Satisfy all the libs dependancies:
  $ sudo aptitude install qt4-dev-tools python-dev python-pyrex\
    libsqlite3-dev python-opengl libmysqlclient-dev

- Check 'qmake' is the Qt4 one:
  $ qmake -v
  Evenually modify the altenatives (if OS ~ Debian)
  $ sudo aptitude install galternatives
  $ gksu galternatives

- Download PyQt 4.3 and unpack it
  $ wget http://www.riverbankcomputing.com/Downloads/PyQt4/GPL/PyQt-x11-gpl-4.3.tar.gz
  $ tar xvf PyQt-x11-gpl-4.3.tar.gz
  (let's say in /home/user/src/PyQt4)

----------------------------------------------------------------
Install
----------------------------------------------------------------

- Copy the contents of the src/ dir in /home/user/src/PyQt4
  (in fact copy pyqt3support.diff and sip/Qt3Support/)

- Apply the pyqt3support.diff in /home/user/src/PyQt4
  $ patch -p0 < pyqt3support.patch

----------------------------------------------------------------
Build
----------------------------------------------------------------

Follow the standard PyQt4 way, build, install

----------------------------------------------------------------
Porting
----------------------------------------------------------------

Start porting you PyQt3 app, and begin your smooth moving
towards PyQt4!
